================================================================================
                            RLS HANDLING
================================================================================

Three-Tier Fallback Strategy:

Tier 1: Try Full Profile
  ↓ Success → Display Data
  ↓ Error → Try Tier 2

Tier 2: Use Fallback Data
  ↓ Display with Warning
  ↓ Page Remains Functional

Tier 3: Optional Minimal Query
  ↓ Fewer Fields
  ↓ Works Around RLS
